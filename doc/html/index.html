What is Perl text to HTML project?

First, Perl is a language for getting your job done. Developed by Larry Wall, who must be by now the most famous programming language for the web age. Almost every little utility can be best and easiest written using odd-looking Perl language. Perl is also a cross platform language, meaning that you can run the programs under Unix, Linux and Windows environments. Learn more about Perl at http://www.perl.org and Windows Perl downloads are available at http://www.activestate.com

Perl program t2html.pl is a spicy little tool that can help turning simple text files into nice looking documents - similar to the white papers. In fact many of the pages in this site has been been built using that little tool. Please navigate to Introduction page above to learn more about the utility. When you have downloaded the program, the best demontration is the self test, which shows you how easy it is to generate web pages. You invoke the program with --test-page and the results are presented below. The sample test page is actually the same for all these test. follow the links to see the original page and how it was rendered according to passed options. 


    Run cmd       : t2html.pl --css-font-normal --Out-dir ../html/ t2html.pl-1.txt
    Original text : t2html.pl-1.txt
    Generated html: t2html.pl-1.html

    Run cmd       : t2html.pl --css-font-readable t2html.pl-2.txt > ../html/t2html.pl-2.html
    Original text : t2html.pl-1.txt
    Generated html: tmp/t2html.pl-2.html

    Run cmd       : t2html.pl --html-frame --css-font-normal --Out-dir doc/html/ doc/examples/t2html.pl-3.txt
    Original text : t2html.pl-3.txt [same as t2html.pl-1.txt]
    Generated html: t2html.pl-3.html
Here are options --css-code-* which affect the CODE section layout by adding the text into a TABLE and then rendering the table background with colors. 

    Run cmd       : t2html.pl --css-code-bg --css-code-note="(?:Notice|Note):' t2html.pl-1.txt > t2html.pl-4.html
    Original text : t2html.pl-1.txt 
    Generated html: t2html.pl-4.html
